# FlaskWX
这是一个基于Flask的WEB项目，通过分析微信网页版的接口，提供对扫码用户的联系人相关信息的数据分析，前端使用Layui，Echarts图表来展示各种数据。
## 使用说明
安装Flask，Requests，等相关库后，运行`runserver.py`文件，启动服务，之后打开浏览器，输入地址，进行扫码展示。
